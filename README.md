# No Fog Perfected

Removes fog from Minecraft, perfected.

Works with both **Fabric** and **NeoForge**.

## Supported Versions

| Minecraft Version | Fabric | NeoForge |
|-------------------|--------|----------|
| 1.21.11           | ✓      | ✓        |
| 26.1.2            | ✓      | ✓        |
| 26.2              | ✓      | ✓        |
